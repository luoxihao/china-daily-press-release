# Rule Conflicts

## Current Finding

No major conflict was found between `新闻采写docx.docx` and the public China Daily corpus observations.

## Alignment

- Both emphasize fact-first structure.
- Both favor concise leads and short paragraphs.
- Both use background after key facts.
- Both support positive but restrained tone.
- Both avoid unsupported praise and empty slogans.
- Both require source-backed quotes and data.

## Tensions To Watch

- Local docx guidance sometimes focuses on Tianjin newsroom workflow and internal contributor lines; use those only when the task is Tianjin-specific or the user asks for local newsroom formatting.
- Public China Daily pages may include publication-specific conventions, photo credits, or editorial modules that should not be copied into a user draft.
- Event releases may use more promotional framing than strict news reports; reduce that tone unless the user explicitly requests promotional copy.

## Resolution Rule

When rules conflict:

1. Protect factual accuracy and source integrity.
2. Follow user-provided source materials.
3. Use docx rules for local newsroom mechanics.
4. Use corpus rules for transferable English news style.
5. Ask for confirmation when the conflict affects a fact, title, quote, or official name.
