# Phrase Rules

## Recommended English Expressions

- drew participants from
- brought together
- featured
- included
- according to the organizer
- the bureau said
- participants joined
- visitors experienced
- the route passed through
- the program aims to
- officials said the event is part of
- the event offered a platform for

## Avoid Or Replace

| Avoid | Prefer |
| --- | --- |
| grandly held | was held / opened |
| successfully held | was held |
| inject vitality | support / add to / contribute to |
| fully demonstrates | shows / reflects / highlights, if sourced |
| a beautiful name card | a tourism brand / a cultural attraction |
| world-renowned without source | well-known / widely visited / source-backed ranking |
| vigorously promote | promote / develop / support |
| unforgettable charm | visitor experience / cultural appeal |
| fruitful results | agreements / projects / outcomes, named specifically |

## Chinese-English Interference Fixes

- `under the background of` -> `against the backdrop of` or delete.
- `in order to further promote` -> `to promote`.
- `relevant departments` -> name the department or write `local authorities`.
- `the masses` -> `residents`, `visitors`, `participants`, or `the public`.
- `rich and colorful activities` -> name the activities.
- `deeply loved by people` -> cite attendance, visitor comments, or omit.
