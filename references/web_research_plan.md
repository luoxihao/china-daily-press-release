# Web Research Plan

This skill already includes an initial public-sample corpus in `china_daily_corpus_notes.md`, collected from public China Daily search results and accessible pages on 2026-06-13. Extend the corpus when a user request concerns a specialized topic.

## Safe Research Method

1. Search public China Daily pages with topic-specific queries.
2. Prefer article pages, not copied reposts.
3. Record title, URL, date, section, topic, article type, structure notes, and style observations.
4. Do not store full article text.
5. Do not bypass website access limits.
6. Do not run high-frequency scraping.
7. Convert observations into transferable rules, not article imitation.

## If Web Access Is Unavailable

- Say that fresh web research could not be completed.
- Use existing local references only.
- Ask the user for article links or source packets.
- Mark new corpus-derived rules as pending.
