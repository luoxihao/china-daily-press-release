# China Daily Press Release Skill

This project-local Codex skill helps draft, rewrite, polish, and fact-check English external-communication news copy with a China Daily-inspired style tendency.

It is not an official China Daily tool and must not claim China Daily publication, approval, authorization, review, or endorsement. Public China Daily articles are used only for style research notes; full article text is not stored.

Use the skill with:

```text
$china-daily-press-release 请根据已核实中文素材生成英文对外传播通稿，并输出事实核查清单和待确认信息。
```

Core references are in `references/`, reusable structures are in `templates/`, and helper checks are in `scripts/`.
